# Basic Sample

这个示例用于快速预览 Celadon Color Engine 的基础能力：

- `theme-day` / `theme-night` 切换
- `celadon-btn` 按钮变体
- `celadon-card` 卡片样式
- `themed-input` 输入表面

## Files

- `index.html`：最小可运行示例

## Usage

如果你只是想快速预览，可以直接打开 `samples/basic/index.html`。

如果你要把它接入业务项目，建议参考根目录 `README.md`，按目标平台引入：

- `index.css`
- `desktop.css` 或 `web-mobile.css` 或 `mobile.css`
