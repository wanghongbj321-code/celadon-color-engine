# Mobile Showcase Sample

This sample presents Celadon Color Engine in a mobile-first layout.

Unlike the broader desktop-oriented showcase in `samples/basic/`, this sample is designed around compact spacing, touch-sized controls, vertical flow, and a phone-like presentation shell.

## Files

- `index.html`: the mobile showcase page

## What This Sample Demonstrates

- `web-mobile.css` as the platform adaptation layer for mobile web
- day/night theme switching in a touch-oriented UI
- semantic token rendering inside a compact mobile layout
- brand accent usage in cards, actions, and status treatment
- touch-friendly forms, controls, and bottom navigation patterns

## Usage

To preview the sample, open `samples/mobile/index.html` in a browser.

For integration guidance, refer to the root `README.md` and import:

- `index.css`
- `web-mobile.css`
